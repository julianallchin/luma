#

# paper-aligned reproduction of Yeung et al., Sony AI 2025). The vendored
# package (next to this requirements.txt) contains the EDM sampler, decoder,
# and log-mel frontend; only third-party deps need installing.
#

# by the `mert` preprocessor, so transformers is no longer pulled in here
# (it's installed for the mert worker's requirements).
#
# torch / torchaudio / torchvision are NOT pinned here — python_env.rs
# strips them at write time so the cu128 GPU wheels installed in step 1
# aren't overwritten by CPU PyPI wheels.
